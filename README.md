
# WhatsApp Backend for Render Deployment

## How to Use

1. Copy `.env.example` to `.env` and fill in:
   - META_TOKEN
   - PHONE_NUMBER_ID

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run locally:
   ```bash
   node server.js
   ```

4. Deploy on [Render.com](https://render.com)
